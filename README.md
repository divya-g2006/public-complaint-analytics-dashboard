# Complaint Analytics Dashboard (MERN)

This repository contains a MERN stack app with:

- `frontend/` React (Vite) + Tailwind
- `backend/` Node.js + Express + MongoDB + JWT

## Local development

1. Backend
   - Copy `backend/.env.example` to `backend/.env` and fill values.
   - Install and run:
     - `npm --prefix backend install`
     - `npm run dev:backend`

2. Frontend
   - Install and run:
     - `npm --prefix frontend install`
     - `npm run dev:frontend`

Backend runs at `http://localhost:5000` by default and the frontend at `http://localhost:5173`.
